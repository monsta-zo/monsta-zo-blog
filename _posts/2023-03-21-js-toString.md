# Object.prototype.toString()

`toString()` 은 객체를 문자열로 바꿔준다.

```js
object.toString();
```

인자는 따로없지만, 객체 내부에서 구현된 `toString()` 메서드가 오버라이드되고 자체적으로 실행된다.

## 2. 예시
