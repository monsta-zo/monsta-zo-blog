# 명시적(Explicit) 형 변환

## Int -> String

*String()*

```js
let a = 777;
let b = String(a);
console.log(b); // "777"
```

## String -> Int

*Number()*

```js
let a = "777";
let b = Number(a);
console.log(b); // 777
```
