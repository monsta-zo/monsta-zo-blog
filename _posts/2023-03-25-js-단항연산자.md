# 단항 연산자

단항 연산자는 하나의 항을 대상으로 연산이 이루어지는 연산자이다. 

## 1. 단항 더하기 ( + )

단항 더하기 연산자는 피연산자 앞에서 사용하며, 숫자가 아닌 변수는 숫자로 변환시킨다.

```js
+true // 1
+false // 0
+null // 0
+[] // 0
+"23" // 23
```
